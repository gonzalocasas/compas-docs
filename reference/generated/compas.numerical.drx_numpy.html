drx_numpy

compas.numerical.drx_numpy(network, factor=1.0, tol=0.1, steps=10000, refresh=100, update=False, callback=None, **kwargs)[source]

Run dynamic relaxation analysis.

Parameters

  • network (obj) – Network to analyse.

  • factor (float) – Convergence factor.

  • tol (float) – Tolerance value.

  • steps (int) – Maximum number of steps.

  • refresh (int) – Update progress every nth step.

  • update (bool) – Update the co-ordinates of the Network.

  • callback (callable) – Callback function.

Returns

  • array – Vertex co-ordinates.

  • array – Edge forces.

  • array – Edge lengths.

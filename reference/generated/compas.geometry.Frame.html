Frame

class compas.geometry.Frame(point=[0, 0, 0], xaxis=[1, 0, 0], yaxis=[0, 1, 0])[source]

Bases: object

The Frame consists of a point and and two orthonormal base vectors.

It represents a plane in three dimensions with a defined origin and orientation.

point

list of float, optional – The origin of the frame. Defaults to [0, 0, 0].

xaxis

list of float, optional – The x-axis of the frame. Defaults to [1, 0, 0].

yaxis

list of float, optional – The y-axis of the frame. Defaults to [0, 1, 0].

Examples

>>> f = Frame([0, 0, 0], [1, 0, 0], [0, 1, 0])
>>> f = Frame.from_points([1, 1, 1], [2, 4, 5], [4, 2, 3])
>>> f = Frame.from_euler_angles([0.5, 1., 0.2])
>>> f = Frame.worldXY()

is_point_on_line_xy

compas.geometry.is_point_on_line_xy(point, line, tol=0.0)[source]

Verify if a point lies on a line in the XY-plane.

Parameters

  • point (sequence of float) – XY(Z) coordinates of a point.

  • line (tuple) – XY(Z) coordinates of two points defining a line.

  • tol (float, optional) – A tolerance. Default is 0.0.

Returns

bool – True if the point is in on the line. False otherwise.

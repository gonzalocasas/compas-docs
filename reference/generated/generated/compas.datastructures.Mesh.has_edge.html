Mesh.has_edge

Mesh.has_edge(u, v, directed=True)[source]

Verify that the mesh contains a specific edge.

Warning

This method may produce unexpected results.

Parameters

  • u (hashable) – The identifier of the first vertex.

  • v (hashable) – The identifier of the second vertex.

  • directed (bool, optional) – Only consider directed edges. Default is True.

Returns

bool – True if the edge exists. False otherwise.

Network.edge_coordinates

Network.edge_coordinates(u, v, axes='xyz')

Return the coordinates of the start and end point of an edge.

Parameters

  • u (hashable) – The key of the start vertex.

  • v (hashable) – The key of the end vertex.

  • axes (str (xyz)) – The axes along which the coordinates should be included.

Returns

tuple – The coordinates of the start point and the coordinates of the end point.

Mesh.get_vertices_attribute

Mesh.get_vertices_attribute(name, value=None, keys=None)

Get the value of a named attribute of multiple vertices.

Parameters

  • name (str) – The name of the attribute.

  • value (object (None)) – The default value.

  • keys (list (None)) – A list of identifiers. Defaults to all vertices.

Returns

values (list) – A list of values of the named attribute of the specified vertices.

Mesh.faces

Mesh.faces(data=False)[source]

Iterate over the faces of the mesh.

Parameters

data (bool, optional) – Return the face data as well as the face keys.

Yields

  • hashable – The next face identifier (key), if data is false.

  • 2-tuple – The next face as a (fkey, attr) tuple, if data is true.

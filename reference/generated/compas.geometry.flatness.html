flatness

compas.geometry.flatness(vertices, faces, maxdev=0.02)[source]

Compute mesh flatness per face.

Parameters

  • vertices (list) – The vertex coordinates.

  • faces (list) – The face vertices.

  • maxdev (float, optional) – A maximum value for the allowed deviation from flatness. Default is 0.02.

Returns

dict – For each face, a deviation from flatness.

Notes

The “flatness” of a face is expressed as the ratio of the distance between the diagonals to the average edge length. For the fabrication of glass panels, for example, 0.02 could be a reasonable maximum value.

Warning

This function only works as expected for quadrilateral faces.

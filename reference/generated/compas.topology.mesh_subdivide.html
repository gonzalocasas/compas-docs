mesh_subdivide

compas.topology.mesh_subdivide(mesh, scheme='tri', **options)[source]

Subdivide the input mesh.

Parameters

  • mesh (Mesh) – A mesh object.

  • scheme ({‘tri’, ‘corner’, ‘catmullclark’, ‘doosabin’}.) – The scheme according to which the mesh should be subdivided.

  • options (dict) – Optional additional keyword arguments.

Returns

Mesh – The subdivided mesh.

Raises

NotImplementedError – If the scheme is not supported.

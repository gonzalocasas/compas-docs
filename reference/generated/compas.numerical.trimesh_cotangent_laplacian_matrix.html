trimesh_cotangent_laplacian_matrix

compas.numerical.trimesh_cotangent_laplacian_matrix(mesh)[source]

Construct the Laplacian of a triangular mesh with cotangent weights.

Parameters

mesh (obj) – The triangular Mesh datastructure object.

Returns

array – The Laplacian matrix with cotangent weights.

Notes

The matrix is constructed such that the diagonal contains the sum of the weights of the adjacent vertices, multiplied by -1. The entries of the matrix are thus

\[\begin{split}\mathbf{L}_{ij} = \begin{cases} - \sum_{(i, k) \in \mathbf{E}_{i}} w_{ik} & if i = j \\ w_{ij} & if (i, j) \in \mathbf{E} \\ 0 & otherwise \end{cases}\end{split}\]

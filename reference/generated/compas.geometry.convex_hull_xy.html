convex_hull_xy

compas.geometry.convex_hull_xy(points, strict=False)[source]

Computes the convex hull of a set of 2D points.

Parameters

points (list) – XY(Z) coordinates of the points.

Returns

list – XY(Z) coordinates of vertices of the convex hull in counter-clockwise order, starting from the vertex with the lexicographically smallest coordinates.

Notes

Implements Andrew’s monotone chain algorithm 1. O(n log n) complexity.
